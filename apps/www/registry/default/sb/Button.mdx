# Button Component

The `Button` component is a versatile component for creating interactive buttons with various styles and behaviors.

**Customization:**

- **Variants:** Choose from different pre-defined variants like "default," "destructive," "outline," and more, each with unique visual styles. (See available variants in Storybook controls.)
- **Sizes:** Adjust the button size using options like "default," "sm," "lg," and "icon." (See available sizes in Storybook controls.)

**Additional Features:**

- **`asChild` prop:** Controls the underlying HTML element. Set it to `true` to render the button as a `Slot` component (useful for customization within layout systems).
