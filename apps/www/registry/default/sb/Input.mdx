# Input Component

An input field allows users to enter text data into a form. It's a versatile component widely used in various applications for user interaction.

**Usage:**

The `Input` component accepts several props:

- **type:** [String] Specifies the input type (e.g., "text", "email", "password"). Defaults to "text".
- **className:** [String] Optional class name to customize the input's appearance.
